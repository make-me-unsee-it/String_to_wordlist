УСЛОВИЕ:
Программа принимает на вход английский текст, произвольной длины, обрабатывает его и выводит все слова,
которые встречаются в тексте (без дубликатов) группируя их по первой букве в алфавитном порядке. Знаки
препинания игнорируются. Регистр заглавных букв не учитывается. Подсчитать ещё и количество вхождений.


    Задача реализована через класс LineToWordList с двумя конструкторами:
    1-й конструктор по умолчанию без принимаемых параметров для получения исходного текста запускает
    сканнер, который реализован в статическом методе scanNoCyr() в классе Scn.
    Сканнер обрабатывает исключение, когда введенный текст содержит кириллицу и при ее обнаружении
    выводит сообщение об ошибке);

    2-й конструктор принимает текст в качестве аргумента типа String при создании экземпляра класса

    Обработка задачи производится последовательным вызовом методов внутри каждого конструктора.
    Алгоритм обработки разбит на 4 этапа (4 protected метода) для удобства тестирования.

    После ввода исходного текста в обоих конструторах первым к нему применяется метод toLowerCase().

    StringBuilder состоящий только из слов, разделенных пробелами, который по шаблону [' '] разделяет
    на отдельные слова которые возвращает в виде одномерного массива типа String
